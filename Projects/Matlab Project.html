MATLAB-Based Simple Pendulum Project

Developed a MATLAB code that inputs parameters of simple pendulum from the user and simulates the system

Project Description

Developed a MATLAB-based project as part of a Computer Applications course, focused on simulating the response of a simple pendulum system. The program reads input parameters from various file types, including Excel and text files, to ensure flexibility and usability. It generates detailed simulations, displaying response graphs for system behavior. Additionally, the application features automated report generation, including images, system descriptions, and results upon user request. This project highlights my skills in MATLAB programming, file handling, and system analysis.
